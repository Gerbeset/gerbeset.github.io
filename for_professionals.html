Open-source Projects I've contributed to:

  • Ryujin — A high-performance, high-order collocation-type finite element solver for conservation laws, including the compressible Navier–Stokes and Euler equations. (GitHub)
    • Implemented implicit-explicit (IMEX) method for the Korteweg de-Vries equation. Included implementing high-order time integration techniques, discrete dispersive operators, and test problems. See "A High-Order IMEX Method for Generalized Korteweg–De Vries Equations" on my list of publications for theoretical details.
    • Implemented implicit-explicit (IMEX) for the Serre-Green-Naghdi equaitons. Included implementing a parallelized, matrix-free elliptic solver which used a geometric multi-grid preconditioner, hyperbolic flux, and test cases. In the process of a writing a paper for this project.
    • Implemented methods for the Korteweg de-Vries equation and the Serre-Green-Naghdi equations. (Note that these methods may not available in the release version; contact the developers or myself if you desire access to the code.)
  • ExactPack — An open-source verification and validation library for generating exact and semi-analytic solutions to benchmark problems in computational physics (LA-CC-14-047, C23033). (GitHub)
    • Developed semi-analytic solutions agnostic to the equation of state and other problem parameters for the Noh Problem.
    • Integrated new solvers into the existing ExactPack framework (see nohblackboxeos)
    • Authored technical reports and a peer-reivew article; see my list of publications, specifically "Semi-Analytic Solutions to the Noh Problem with a Black Box Equation of State."
